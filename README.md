# Web API using JWT token Authorization

This project serves as a backend service designed to have sample data for testing purpose. It follows modern architectural patterns and ensures that sensitive data is protected through industry-standard token-based security. This is a robust, scalable RESTful API built with ASP.NET Core, featuring secure authentication and authorization via JWT (JSON Web Tokens). 

This project uses JSON files from [Chaitrali JSON](https://github.com/Chaitrali-tambe/ChaitraliJson) and includes following data in json format.

|   Data         | Description                                   |
| -------------- | --------------------------------------------- |
| Products       | 40 Product Items with details                 |
| Orders         | 40 Orders with details                        |
| Customers      | 19 Customers with details                     |
| Location       | 19 Locations with details                     |
| Categories     | 3 Categories with their Subcategories below   |
| Transactions   | 40 Transactions and their details             |
